# TSA KPI Dashboard — How to View

## Quick Access
1. Open the shared Google Drive folder
2. Click on **RACCOONS_KPI_DASHBOARD.html**
3. Google will show a preview — click **"Open with"** at the top and choose **"Open in new window"** or download it
4. The dashboard opens in your browser with full interactivity

## Best Experience
For the full interactive experience (tooltips, filters, charts):
1. **Download** the HTML file to your computer
2. **Double-click** to open in your default browser (Chrome, Edge, Firefox)
3. That's it — everything works offline, no server needed

## What You'll See
- **Member Cards**: Your personal stats (On Time, Late, Accuracy)
- **ETA Accuracy**: Heatmap showing who delivers on time per week
- **Execution Time**: How fast tasks are completed
- **Reliability**: Implementation quality (when rework labels are active)
- **Team Activity**: Task volume per person per week

## Filters
- **Person**: Filter by team member
- **Category**: Internal vs External (client) work
- **Segment**: All / Internal / External toggle

## How It Updates
The dashboard is refreshed daily with the latest data from Linear.
Each update replaces the file in this folder — your link always shows the latest version.
